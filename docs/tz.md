# **ТЕХНИЧЕСКОЕ ЗАДАНИЕ (ТЗ)**

## **Проект: F.O.C.U.S — AI-Навигатор Профессий**

---

# 1. **Цель проекта**

Создать цифровую платформу, которая с помощью ИИ анализирует личность и навыки пользователя, определяет подходящие профессии и формирует персональную карьерную дорожную карту (roadmap) и учебный план.

---

# 2. **Основная задача платформы**

1. Провести психометрический тест.
2. Определить сильные и слабые стороны пользователя.
3. Подобрать профессии, подходящие под психотип, интересы и навыки.
4. Сформировать roadmap: Junior → Middle → Senior → Expert.
5. Подобрать персональные учебные материалы.
6. Показать востребованность профессий на рынке труда.
7. Обновлять рекомендации по мере прогресса пользователя.

---

# 3. **Функциональные модули**

## 3.1. **Модуль регистрации и профиля**

* Регистрация (email/телефон/Google).
* Личный кабинет.
* История тестов.
* Сохранение roadmap.
* Обновление данных пользователя (интересы, цели).

---

## 3.2. **Психометрический тест**

* 30–60 вопросов.
* Модели: Big Five + RIASEC + дополнительные вопросы по интересам.
* Автоматическая оценка результатов.
* Генерация профиля: психотип, личностные особенности, интересы, soft skills.

---

## 3.3. **ИИ-Аналитика личности**

* Генерация **personality embedding** (вектор пользователя).
* Определение сильных и слабых сторон.
* Определение типа обучения (визуал, практик, логик и т.д.).
* Генерация короткого текстового отчёта.

---

## 3.4. **Подбор профессий**

На основе профиля:

* 3–5 подходящих профессий.
* Краткое описание каждой профессии.
* Почему она подходит.
* Требуемые навыки.
* Прогноз востребованности в Узбекистане.

---

## 3.5. **Генерация Roadmap**

Структура:

* **Этап Junior (0–3 месяца)**
* **Этап Pre-Middle (3–6 месяцев)**
* **Этап Middle (6–12 месяцев)**
* **Этап Senior-потенциал (12–24 месяцев)**

Для каждого этапа:

* Hard skills.
* Soft skills.
* Практические задания.
* Портфолио.
* Стажировки/конкурсы.

---

## 3.6. **Подбор учебных материалов**

ИИ подбирает:

* курсы,
* статьи,
* видео,
* задания.

Контент фильтруется:

* по уровню знаний,
* по стилю обучения,
* по интересам пользователя.

---

## 3.7. **Аналитика рынка труда**

* Популярность профессии в Узбекистане.
* Примерные зарплаты.
* Требуемые навыки.
* Ссылки на вакансии (hh.uz, другие порталы).

---

## 3.8. **ИИ-помощник**

* Ответы на вопросы пользователя.
* Объяснение непонятных тем.
* Персональные советы по обучению.
* Пошаговые подсказки по roadmap.

---

# 4. **Технические требования**

## 4.1. **Frontend**

* Next.js / React
* TypeScript
* Tailwind CSS
* Адаптивная вёрстка (телефон/планшет/ПК)

---

## 4.2. **Backend**

Вариант 1: Node.js (NestJS)
Вариант 2: Python (FastAPI)

Функции:

* API для тестов
* API для профиля
* API для аналитики
* Авторизация (JWT)
* Генерация отчётов

---

## 4.3. **База данных**

PostgreSQL
Таблицы:

* users
* test_results
* personality_profiles
* professions
* milestones (roadmap)
* user_progress
* recommended_content

---

## 4.4. **ИИ-модули**

1. **LLM (OpenAI/Gemini/Mistral)**

   * анализ текстов,
   * генерация рекомендаций,
   * генерация roadmap.

2. **Embeddings**

   * PGVector
   * сравнение профиля и профессий.

3. **Модуль адаптивного обучения**

   * анализ прогресса
   * рекомендации следующих шагов

---

## 4.5. **Интеграции**

* Вакансии: hh.uz (по API или парсинг)
* YouTube/курсы (фильтрация по профессиям)

---

# 7. **Ожидаемый результат**

Готовая AI-платформа, которая:

* проводит анализ личности,
* подбирает подходящие профессии,
* создаёт персональный roadmap,
* предлагает материалы обучения,
* показывает востребованность профессий,
* помогает молодёжи выбрать карьеру и найти работу
