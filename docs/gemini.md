Ниже я подготовил стратегию **экстренного создания MVP** специально для прохождения во второй этап, фокусируясь на критериях оценки (Ясность — 25%, Связь с проблемой — 25%, Техническая глубина — 20%).

### 1. Какой формат MVP готовим?

**Главное правило хакатона:** Не пытайтесь реализовать всё ТЗ. Реализуйте "Счастливый путь" (Happy Path) — один идеальный сценарий использования от начала до конца.

Для проекта **F.O.C.U.S** MVP должен выглядеть так:
1.  **Лендинг/Вход:** Простая страница с кнопкой "Начать путь".
2.  **Тестирование (Упрощенное):** Вместо 60 вопросов сделайте 5-7 вопросов (имитация) или кнопку "Заполнить автоматически" (для демо).
3.  **Магия AI (Loader):** Красивая анимация "ИИ анализирует ваш профиль...".
4.  **Результат (Вау-эффект):**
    *   Карточка: "Твоя идеальная профессия: Backend Developer".
    *   Генерация Roadmap (визуальная шкала Junior -> Senior).
    *   Список (3 шт.) рекомендуемых курсов.
5.  **Чат-бот (Бонус):** Виджет в углу экрана.

**Статус проекта для описания:** Указываем **"Prototype"** или **"MVP"** (если успеете связать с реальным API).

---

### 2. План быстрой реализации (Стек: Next.js + OpenAI API)

Я рекомендую использовать **Next.js (React)**, так как он позволяет сделать и фронтенд, и бэкенд (API routes) в одном проекте, что экономит время.

#### Шаг 1. Страница `/demo` (Обязательное требование)
Вам нужно создать роут `/demo` на вашем сайте.
*   **Контент:** Заголовок проекта, встроенный видеоплеер (YouTube/Vimeo), текстовое описание (по структуре из PDF), ссылка на сам апп (например, кнопка "Попробовать демо").

#### Шаг 2. Реализация ядра (AI-Роадмап)
Не делайте сложную базу данных PostgreSQL сейчас, если её еще нет. Используйте **Mock-данные** или простой **Local Storage**, а запросы отправляйте сразу в OpenAI.

*   **Промпт для ИИ:**
    > "Ты карьерный консультант. Пользователь ответил на вопросы: [Ответы]. Его психотип: [Тип]. Предложи 1 профессию, дай описание и создай пошаговый Roadmap (Junior, Middle, Senior) с темами для изучения. Верни ответ в формате JSON."

*   **Визуализация:** Полученный JSON отрисуйте в виде красивого таймлайна. Это покажет "Техническую глубину" и работу AI.

#### Шаг 3. Бонусные задания (Легкие баллы)
*Обязательно сделайте их, это повысит шансы.*

*   **BONUS 1: Чат-бот**
    *   Вставьте простую иконку чата.
    *   При нажатии открывается окно.
    *   Сделайте API-роут `/api/chat`.
    *   В системный промпт зашейте: "Если спрашивают 'Что делает проект?', отвечай: 'Наш проект F.O.C.U.S помогает...'".
    *   *Хак:* Для демо можно даже сделать hard-coded ответы на 2 обязательных вопроса, если не успеваете подключить LLM к чату. Но лучше подключить (это 10 строк кода с OpenAI SDK).

*   **BONUS 2: API Access**
    *   Создайте страницу `/api-docs` или просто блок на странице `/demo`.
    *   Покажите пример запроса (как в PDF):
        ```json
        POST /api/generate-roadmap
        { "skills": ["python", "sql"], "interest": "data" }
        ```
    *   И реальный ответ JSON. Это докажет жюри, что у вас "AI-powered endpoint".

### 3. Сценарий Демо-видео (1-3 минуты)

Видео — это 50% успеха (25% ясность + 25% связь с проблемой).

*   **0:00 - 0:40 (Питч):**
    *   *Голос:* "В Узбекистане тысячи студентов не знают, кем стать. Они тратят годы на обучение не тому. Мы — команда [Название], представляем F.O.C.U.S — AI-навигатор, который строит карьеру, а не просто дает советы."
*   **0:40 - 1:30 (Демо процесса):**
    *   Показываем экран. "Пользователь проходит тест (быстро прокликиваем)."
    *   "Наши алгоритмы на базе LLM анализируют психотип."
    *   *Акцент:* "Смотрите, система сгенерировала персональный Roadmap. Это не шаблон, это уникальный путь под пользователя."
*   **1:30 - 2:00 (Технологии и Бонусы):**
    *   "Мы используем Next.js и OpenAI Embeddings."
    *   "У нас есть встроенный AI-ментор (показываем чат-бот, задаем вопрос 'Для кого этот продукт?')."
    *   "Наша система открыта для интеграций через API (показываем JSON ответ)."
*   **2:00 - 2:30 (Заключение):**
    *   "F.O.C.U.S решает проблему безработицы среди молодежи. Спасибо."
